# TW2_ba

日本人小児の TW2-RUS スコアと骨年齢を計算する静的Webページです。

公開ページ:
[![Open GitHub Pages](https://img.shields.io/badge/Open-GitHub%20Pages-2ea44f?style=for-the-badge)](https://satashun.github.io/TW2_ba/)

## 使い方

1. 公開ページを開き、`男児` または `女児` を選択します。
2. RUS 13骨それぞれについて、画像所見に対応するステージをプルダウンから選択します。
3. 入力に応じて、`RUS 合計スコア` と `骨年齢` が自動で更新されます。
4. `スコア内訳` で各骨の選択ステージと点数を確認します。
5. 最初からやり直す場合は `リセット`、記録を残す場合は `印刷` を使います。

## 注意

- 本ツールは日本人標準値に基づく補助計算用です。
- 最終的な判定は TW2 アトラスおよび原典の標準値表で確認してください。
